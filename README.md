# Ebookcoin Web Client

亿书：去中心化的自出版和版权保护应用

This repository is a submodule of the Ebookcoin, and bases on [Ember.js](https://emberjs.com/).

## Installation

Install node modules:

```
npm install
```

Install bower components:

```
bower install
```

Build the user-interface:

```
grunt release
```
